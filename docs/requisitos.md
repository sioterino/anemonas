### 1. Requisitos Funcionais
Os requisitos funcionais (RF) descrevem as funcionalidades do sistema, e devem servir de guia para montar o backlog do projeto.

| Código | Descrição                                                                                              |
| ------ | ------------------------------------------------------------------------------------------------------ |
| RF01   | Cadastro e autenticação de usuários com diferentes papéis: aluno, monitor e coordenador/administrador. |
| RF02   | Sistema de login seguro com autenticação diferenciada por tipo de usuário.                             |
| RF03   | Chat monitorado para comunicação entre usuários, com possibilidade de moderação por IA.                |
| RF04   | Fórum interativo com sistema de votação para destacar as melhores respostas.                           |
| RF05   | Agenda pública de disponibilidade dos monitores.                                                       |
| RF06   | Agendamento de monitorias individuais ou em grupo.                                                     |
| RF07   | Filtros de busca de mentor por matéria e histórico de atendimentos para facilitar o agendamento.       |
| RF08   | Repositório de questões categorizadas por matéria e nível de dificuldade.                              |
| RF09   | Trilhas de aprendizado com quizzes interativos e feedback imediato.                                    |
| RF10   | Ferramentas para que monitores criem e indiquem exercícios específicos.                                |
| RF11   | Visualização de desempenho individual dos alunos.                                                      |
| RF12   | Estatísticas de engajamento em fóruns e atividades.                                                    |
| RF13   | Relatórios de uso da plataforma para monitores e administradores.                                      |
| RF14   | Notificações automáticas por e-mail para lembretes, atividades e monitorias.                           |
| RF15   | Sistema de recomendação de conteúdos, atividades e monitorias com base no uso da plataforma.           |
